## Claude Codeって何？

**ターミナル（黒い画面）で動くAIアシスタント**です。

ChatGPTやClaude.aiとの違い：
- ファイルを直接読み書きできる
- コマンドを実行できる
- プロジェクト全体を理解して作業できる

つまり、**AIが実際に手を動かしてくれる**ツールです。

## 必要なもの

| 必要なもの | 確認方法 |
|-----------|----------|
| Mac | 今使っているPCがMacならOK |
| Anthropicアカウント | [claude.ai](https://claude.ai) で登録 |
| 有料プラン | Pro($20/月) または Max($100/月) |

### Node.jsの確認

ターミナルを開いて、以下を入力：

```bash
node -v
```

`v18.0.0` 以上の数字が出ればOK。
「command not found」と出たら、[Node.jsのインストール](#nodejs-のインストール) へ。

## インストール（3ステップ）

### ステップ1: ターミナルを開く

1. `Command + Space` でSpotlightを開く
2. 「ターミナル」と入力
3. Enter

### ステップ2: Claude Codeをインストール

ターミナルに以下をコピー＆ペースト：

```bash
npm install -g @anthropic-ai/claude-code
```

### ステップ3: 起動して認証

```bash
claude
```

ブラウザが開くので、Anthropicアカウントでログイン。
これで完了！

## 最初の一歩

### 1. プロジェクトフォルダに移動

```bash
cd ~/Documents/my-project
```

（`my-project` は自分のフォルダ名に置き換え）

### 2. Claude Codeを起動

```bash
claude
```

### 3. 話しかけてみる

```
このフォルダの中身を説明して
```

AIがファイルを読んで説明してくれます。

### 4. 何かお願いしてみる

```
READMEファイルを作って
```

AIが実際にファイルを作成します。

## これだけ覚えればOK

### 基本操作

| やりたいこと | 入力する内容 |
|-------------|-------------|
| 終了する | `Ctrl + C` または `/exit` |
| 履歴をクリア | `/clear` |
| ヘルプを見る | `/help` |

### 便利なスラッシュコマンド

| コマンド | 効果 |
|---------|------|
| `/init` | プロジェクト設定ファイル(CLAUDE.md)を自動生成 |
| `/compact` | 会話を要約してメモリ節約 |
| `/cost` | 今回のセッションのコスト確認 |

### 話しかけ方のコツ

**具体的に指示する**：

```
❌ 「このコードを直して」
⭕ 「このコードの認証エラーを修正して」
```

**複数の視点で検討させる**：

```
「5人の専門家視点で、この設計を評価して」
```

**並列で作業させる**：

```
「並列サブエージェントで、以下を同時に調査して：
- 競合A社の製品
- 競合B社の製品
- 市場トレンド」
```

## よくある質問

### Q: 料金はどれくらいかかる？

セッション中に `/cost` で確認できます。
目安として、1時間の作業で $0.5〜$2 程度（使い方による）。

### Q: 間違ったことをしたら？

`Ctrl + C` で操作をキャンセルできます。
ファイルを壊してしまっても、Gitを使っていれば戻せます。

### Q: 途中で止まったら？

`Enter` を押すか、`続けて` と入力すると再開します。

### Q: 長い作業中に接続が切れたら？

```bash
claude --resume
```

で前回のセッションを再開できます。

### Q: もっと詳しく知りたい

## Node.js のインストール

Node.jsがない場合は、以下のどちらかでインストール：

### 方法1: Homebrewを使う（推奨）

```bash
# Homebrewがなければ先にインストール
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Node.jsをインストール
brew install node
```

### 方法2: 公式サイトからダウンロード

1. [nodejs.org](https://nodejs.org) にアクセス
2. 「LTS」と書かれたボタンをクリック
3. ダウンロードしたファイルを開いてインストール
